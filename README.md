# Trajectory 弹道轨迹
 2019 SCU-CS 研究与开发实战课程作业   
 一个模仿 __4399弹弹堂__ 与 __TNT弹道轨迹__ 的2D游戏, 使用 EazyX引擎 完成  
 EazyX 官网地址: https://easyx.cn/  
 EazyX 文档地址: https://docs.easyx.cn/zh-cn/intro

 # 开发环境搭建
 依次按照如下步骤安装
 - 安装 vs2019 或其他 EazyX 支持的IDE
 - 使用官网自动安装程序安装 EazyX   
 - 从 github 上 clone 该项目到本地
